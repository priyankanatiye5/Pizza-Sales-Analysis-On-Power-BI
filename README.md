# Pizza-Sales-Analysis-On-Power-BI
This project delivers an interactive Pizza Sales Dashboard built in Power BI to analyze revenue performance, customer ordering behavior, and product-level insights.  The dashboard transforms raw sales data into actionable insights to support business decisions in pricing, inventory, and product strategy


🎯 Objective 

The goal of this project is to answer key business questions:

Which pizzas generate the most revenue vs volume?

When are peak sales periods?

Which categories and sizes drive performance?

What are the best and worst-selling products?



📁 Dataset

The dataset contains:

Order ID

Order Date & Time

Pizza Name & Category

Pizza Size

Quantity

Price



⚙️ Data Preparation

Performed in Power Query:

Removed null values and inconsistencies

Standardized column formats

Created derived columns (Day, Month, Hour)

Ensured data type accuracy
